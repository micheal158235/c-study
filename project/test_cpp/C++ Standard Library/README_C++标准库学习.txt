库归类说明
Utilities              :工具库
Concepts               :概念库
Coroutines             :协程库
DynamicMemoryManagement:动态内存管理库
NumericLimits          :数值界限库
ErrorHandling          :错误处理库
Strings                :字符串库
Containers             :容器库
Iterators              :迭代器库
Ranges                 :范围库
Algorithms             :算法库
Numerics               :数值库
IO                     :输入/输出库
Localization           :本地化库
RegularExpressions     :正则表达式库
AtomicOperations       :原子操作库
ThreadSupport          :线程支持库
Filesystem             :文件系统库

文件件夹命名规则
如cstdlib_c98_Utilities
cstdlib  :库名
c98      :该库在c98标谁开始加入
Utilities:该库归类到工具库
